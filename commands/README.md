# Commands

This directory contains optional workflow entry points that orchestrate canonical AIPOM skills.

Commands must not duplicate skill content or become an alternative source of truth.

## Starting Paths

- [Assess an Organization](aipom-assess-organization.md)
- [Evaluate an AI Initiative](aipom-evaluate-initiative.md)
- [Redesign a Product-Team Workflow](aipom-redesign-workflow.md)

Each command routes users through canonical skills while preserving evidence, assumptions, decisions, ownership, and unresolved questions between steps.

# Skills

This directory is the canonical source for AIPOM skill content.

Each skill lives at `skills/<skill-name>/SKILL.md` and follows [the skill specification](../docs/SKILL-SPEC.md).

The library currently contains 34 active skills across the completed Phase 1, Phase 2, and Phase 3 releases:

- Phase 1 establishes the assessment-to-action operating-model foundation.
- Phase 2 converts findings into repeatable practices across strategy, portfolio, workflows, context and data, governance, and capability.
- Phase 3 adds economics, dependency, context lifecycle, evaluation, incident control, readiness, and operating-model design infrastructure.

Start with the generated [skill catalog](../catalog/INDEX.md) to browse by category, type, or phase. Use [ROADMAP.md](../ROADMAP.md) for canonical phase status and the remaining Phase 3 and Phase 4 plan.

Every active skill includes a canonical `SKILL.md`, a reusable template, a synthetic worked example, and a weak example. Generated platform packages belong in `dist/`; do not add platform-specific instructions to the canonical skills.

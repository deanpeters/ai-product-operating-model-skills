# Synthetic Worked Example: Case-Brief Platform

The audit finds that prompts and evaluation cases are portable, but citations depend on a proprietary retrieval API and logs cannot be exported at required detail. A 30% price increase has modest impact; loss of regional processing blocks operation. A second model does not solve the retrieval concentration. The team constrains scale, negotiates log export and notice terms, prototypes a portable retrieval interface, and sets a renewal trigger based on successful migration rehearsal.
